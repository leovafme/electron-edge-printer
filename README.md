
> Module for implementing printer functionality for windows using electron-edge

## Install

```
$ npm install electron-edge-printer
```
## Example Setup

```
Inside Client.js
Set <UrlToLoad> to web page path in 
window.loadURL(<UrlToLoad>);

$ cd Examples\Client\code\

$ npm install 

$ bower install 

$ npm start 

```
